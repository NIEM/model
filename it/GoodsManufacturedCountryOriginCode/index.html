it:GoodsManufacturedCountryOriginCode

element GoodsManufacturedCountryOriginCode in namespace http://release.niem.gov/niem/domains/internationalTrade/4.1/

Definition

A code to identify the country in which the goods have been produced or manufactured, according to criteria laid down for the application of the Customs tariff or quantitative restrictions, or any measure related to trade.
