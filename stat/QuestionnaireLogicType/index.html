stat:QuestionnaireLogicType

complexType QuestionnaireLogicType in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

A data type for a sequence of questions, question blocks and statements based on factors such as the current location, the response to the previous questions etc., invoking navigation and validation rules to apply.
