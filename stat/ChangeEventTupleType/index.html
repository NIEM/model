stat:ChangeEventTupleType

complexType ChangeEventTupleType in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

A data type that records which identifiable artifacts were changed by a change event. It keeps track of the source Identifiable artifact(s) to which the change event was applied and the resulting target Identifiable artifact(s).
