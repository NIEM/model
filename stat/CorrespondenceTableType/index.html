stat:CorrespondenceTableType

complexType CorrespondenceTableType in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

A data type for a correspondence table expresses the relationship between two statistical classifications. These are typically: two versions from the same classification series; statistical classifications from different classification series; a variant and the version on which it is based; or, different versions of a variant. In the first and last examples, the correspondence table facilitates comparability over time. Correspondence relationships are shown in both directions.
