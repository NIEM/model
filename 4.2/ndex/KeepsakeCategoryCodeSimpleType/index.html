ndex:KeepsakeCategoryCodeSimpleType

simpleType KeepsakeCategoryCodeSimpleType in namespace http://release.niem.gov/niem/codes/fbi_ndex/4.2/

Definition

A data type for a code that identifies items collected as a hobby or given in friendship.
