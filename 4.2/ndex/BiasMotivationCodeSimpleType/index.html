ndex:BiasMotivationCodeSimpleType

simpleType BiasMotivationCodeSimpleType in namespace http://release.niem.gov/niem/codes/fbi_ndex/4.2/

Definition

A data type for a code that identifies the type of bias that motivated the offense, if any. Includes all NIBRS codes, plus additional codes.
