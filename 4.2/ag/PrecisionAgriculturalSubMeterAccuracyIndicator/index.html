ag:PrecisionAgriculturalSubMeterAccuracyIndicator

element PrecisionAgriculturalSubMeterAccuracyIndicator in namespace http://release.niem.gov/niem/domains/agriculture/4.0/

Definition

True if geospatial data collected from Precision Agriculture technology has horizontal coordinate measurements that are accurate within a sub meter; false otherwise.
