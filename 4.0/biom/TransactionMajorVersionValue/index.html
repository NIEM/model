biom:TransactionMajorVersionValue

element TransactionMajorVersionValue in namespace http://release.niem.gov/niem/domains/biometrics/4.0/

Definition

A current major version number of the ANSI/NIST-ITL standard implemented by the software or system creating the file. Always "05" for this version of the standard
