j:ArresteeClearanceIndicator

element ArresteeClearanceIndicator in namespace http://release.niem.gov/niem/domains/jxdm/6.0/

Definition

True if the arrest of this person clears the Incident; false if the Incident was cleared prior to the arrest of this person.
