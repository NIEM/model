j:HearingTimeConstraintsMetIndicator

element HearingTimeConstraintsMetIndicator in namespace http://release.niem.gov/niem/domains/jxdm/6.0/

Definition

True if all time constraints have been met and all reports and evidence were submitted 24 hours prior to the hearing; false otherwise.
