em:NotifierRoleDescriptionText

element NotifierRoleDescriptionText in namespace http://release.niem.gov/niem/domains/emergencyManagement/4.0/

Definition

A description of the message Originator's/Recipient's role, as it may determine message distribution and presentation decisions. The list and associated value(s) is derived from the Uniform Resource Name of a published list of values and definitions, and the content is a string (which may represent a number) denoting the value itself
