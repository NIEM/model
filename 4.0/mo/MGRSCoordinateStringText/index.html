mo:MGRSCoordinateStringText

element MGRSCoordinateStringText in namespace http://release.niem.gov/niem/domains/militaryOperations/4.0/

Definition

A complete coordinate string from the Military Grid Reference System (MGRS) which represents a location with a Universal Transverse Mercator (UTM) or Univeral Polar Steriographic (UPS) coordinate and a unique military grid square.
