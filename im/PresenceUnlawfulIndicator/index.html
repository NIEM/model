im:PresenceUnlawfulIndicator

element PresenceUnlawfulIndicator in namespace http://release.niem.gov/niem/domains/immigration/5.0/

Definition

True if the District Officer finds that the alien student is in status violation, or has been denied a reinstatement; false otherwise.
