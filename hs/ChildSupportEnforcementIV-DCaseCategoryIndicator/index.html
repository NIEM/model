hs:ChildSupportEnforcementIV-DCaseCategoryIndicator

element ChildSupportEnforcementIV-DCaseCategoryIndicator in namespace http://release.niem.gov/niem/domains/humanServices/5.0/

Definition

True if the child support enforcement case implicates Title IV-D of the Social Security Act (e.g., the custodial parent has received funds from the Temporary Assistance for Needy Families (TANF) program); false otherwise.
