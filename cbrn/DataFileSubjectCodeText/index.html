cbrn:DataFileSubjectCodeText

element DataFileSubjectCodeText in namespace http://release.niem.gov/niem/domains/cbrn/5.0/

Definition

A description of a kind of subject matter recorded in a digital data file. If the kind is Other, a description should be provided in BinaryDescriptionText.
