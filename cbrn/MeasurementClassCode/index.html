cbrn:MeasurementClassCode

element MeasurementClassCode in namespace http://release.niem.gov/niem/domains/cbrn/5.0/

Definition

An indicator A code indicating whether the data are a measurement of an item (Foreground), an environmental background (Background), a calibration source (Calibration), the intrinsic activity of the radiation measurement instrument (IntrinsicActivity), or not specified (NotSpecified).
