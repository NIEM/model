cbrn:DetectionEventInstrumentDataAbstract

element DetectionEventInstrumentDataAbstract in namespace http://release.niem.gov/niem/domains/cbrn/4.0/

Definition

A data concept for the set of data output by an instrument for a detection event. This includes the reported measurement and analysis data, information about the instrument, and information about the item(s) which it is measuring/inspecting.
