cbrn:DataFileSetUUID

element DataFileSetUUID in namespace http://release.niem.gov/niem/domains/cbrn/5.0/

Definition

A Unique identifier of a file set. Also serves as the Data File UUID of a file containing the identifiers of the members of the file set.
