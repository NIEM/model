cbrn:yUncertaintyValue

attribute yUncertaintyValue in namespace http://release.niem.gov/niem/domains/cbrn/4.2/

Definition

A value for the 1-sigma absolute total uncertainty (i.e., including all sources of uncertainty) in the Y value of an equation. An uncertainty value of 0 means unknown.
