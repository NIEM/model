cbrn:AnalysisResultStatusCode

element AnalysisResultStatusCode in namespace http://release.niem.gov/niem/domains/cbrn/4.0/

Definition

A description of the the success or failure status of a measurement analysis. If this element is omitted, the analysis is considered successful. The AnalysisResultDescription element shall be used to describe an analysis failure in detail.
