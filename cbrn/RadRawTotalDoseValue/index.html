cbrn:RadRawTotalDoseValue

element RadRawTotalDoseValue in namespace http://release.niem.gov/niem/domains/cbrn/5.0/

Definition

A Total Dose measurement data element(s) used to produce derived data. There shall be no duplicate IDREF values in the list. This is required whenever the element is used within a Derived Data block and prohibited otherwise.
