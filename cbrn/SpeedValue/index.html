cbrn:SpeedValue

element SpeedValue in namespace http://release.niem.gov/niem/domains/cbrn/5.0/

Definition

An object's speed (e.g., radiation measurement instrument, radiation detector, or measured item). If an orientation bearing is defined by the presence of the Orientation element, then the SpeedValue is considered to be along this bearing.
