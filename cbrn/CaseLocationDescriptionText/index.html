cbrn:CaseLocationDescriptionText

element CaseLocationDescriptionText in namespace http://release.niem.gov/niem/domains/cbrn/4.0/

Definition

A description of the locale or location associated with a case when it was initiated. For a case that is a collection of cases, may describe a route or involved locations/locales.
