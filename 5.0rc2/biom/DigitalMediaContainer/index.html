biom:DigitalMediaContainer

element DigitalMediaContainer in namespace http://release.niem.gov/niem/domains/biometrics/5.0/

Definition

A kind of container which contains information about the container format that encapsulates the audio data of the electronic file used to carry the voice data in the digital recording.
