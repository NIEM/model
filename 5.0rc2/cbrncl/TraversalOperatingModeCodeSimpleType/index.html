cbrncl:TraversalOperatingModeCodeSimpleType

simpleType TraversalOperatingModeCodeSimpleType in namespace http://release.niem.gov/niem/codes/cbrncl/5.0/

Definition

A data type for the operating modes of a traversal.

Diagram

Enumerations

ValueDefinition
ExerciseThe traversal is a live encounter conducted for purposes of evaluation of processes, procedures, and equipment.
OperationalThe traversal is a real encounter.
SimulatedThe traversal is a simulation only. It does not involve real assets or personnel.
TestThe traversal is a live encounter conducted for test purposes only.
TrainingThe traversal is a live encounter conducted for training purposes only.
