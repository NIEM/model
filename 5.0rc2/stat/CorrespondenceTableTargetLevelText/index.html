stat:CorrespondenceTableTargetLevelText

element CorrespondenceTableTargetLevelText in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

A correspondence is normally restricted to a certain level in the target Statistical classification. In this case, source items are assigned only to target items on the given level. If no level is indicated, source items can be assigned to any level of the target statistical classification.
