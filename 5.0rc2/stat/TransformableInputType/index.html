stat:TransformableInputType

complexType TransformableInputType in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

A data type for a type of process input whose content goes into a process step and is changed in some way by the execution of that process step. Some or all of the content will be represented in the transformed output.
