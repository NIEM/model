stat:DomainSentinelIndicator

element DomainSentinelIndicator in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

True if the domain is sentinel (i.e. values used to represent a state in the processing life-cycle e.g. missing data); false if the domain is substantive (i.e. values used to represent an observation of some Unit of interest).
