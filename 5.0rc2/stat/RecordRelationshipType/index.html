stat:RecordRelationshipType

complexType RecordRelationshipType in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

A data type for a set of relationships between logical records within a unit data structure. It must have both a source logical record and a target logical record in order to define the relationship.
