stat:MeasurementClassificationType

complexType MeasurementClassificationType in namespace http://release.niem.gov/niem/auxiliary/statistics/5.0/

Definition

A data type for a type of a measure e.g. mass or currency. The measurement type groups all measurement units, which can be converted into each other. A measurement type can have a standard measurement unit, which can be used for conversion between different measurement units.
