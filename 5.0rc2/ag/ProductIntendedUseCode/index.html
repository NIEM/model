ag:ProductIntendedUseCode

element ProductIntendedUseCode in namespace http://release.niem.gov/niem/domains/agriculture/5.0/

Definition

A product code identifying the intended use for an agricultural product such as a crop/commodity, tree, livestock, etc. Codes for intended uses are specific to a particular combination of Core Product Code and Core Product Type Code and are maintained by the Pilot CVT (Crop Validation Table).
