j:CourtOrderOutOfStateIndicator

element CourtOrderOutOfStateIndicator in namespace http://release.niem.gov/niem/domains/jxdm/7.0/

Definition

True if the subject of an order is either out of state, or the order is in effect for agencies in other states; false otherwise.
