j:SubjectMO

element SubjectMO in namespace http://release.niem.gov/niem/domains/jxdm/7.0/

Definition

A set of details pertaining to facts attained from an actual crime scene, giving investigators insights into the subjects method of working or operating, actions used to execute the crime, prevent detection, and/or avoid apprehension.
