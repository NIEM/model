im:PL94-484AmendedIndicator

element PL94-484AmendedIndicator in namespace http://release.niem.gov/niem/domains/immigration/5.0/

Definition

True if the Exchange Visitor (EV) is participating as an alien Physician (Category code of 6), which would result in the EV being flagged as subject to the two-year residence requirement; false otherwise.
