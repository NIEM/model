ag_codes:ProductIntendedUseSimpleType

simpleType ProductIntendedUseSimpleType in namespace http://release.niem.gov/niem/codes/ag_codes/4.0/

Definition

A data type identifying the intended use code for an agricultural product such as a crop/commodity, tree, livestock, etc. Codes for intended uses are specific to a particular combination of Core Product Code and Core Product Type Code and are maintained by the Pilot CVT (Crop Validation Table).
