j:RegistrationOutsideJurisdictionIndicator

element RegistrationOutsideJurisdictionIndicator in namespace http://release.niem.gov/niem/domains/jxdm/7.0/

Definition

True if a property item is found to be registered outside of the immediate jurisdiction of an agency when processed; false otherwise.
