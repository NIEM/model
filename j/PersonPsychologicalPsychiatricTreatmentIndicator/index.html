j:PersonPsychologicalPsychiatricTreatmentIndicator

element PersonPsychologicalPsychiatricTreatmentIndicator in namespace http://release.niem.gov/niem/domains/jxdm/7.0/

Definition

True if a person has been evaluated or treated for psychological/psychiatric concerns, e.g., anger management; behavioral concerns; cognitive concerns, including concerns related to suicide; false otherwise.
